# LiveOps Production Calendar

## Cadence

- Small update every 2 weeks.
- Major update every 6-8 weeks.

## Two-Week Small Update Template

- Week 1
  - Add one quest chain variant.
  - Add one limited weather profile.
  - Add 1-2 cosmetic rewards.
- Week 2
  - KPI review and balancing patch.
  - Community prompt for next season vote.

## 6-8 Week Major Update Template

- Sprint A
  - New biome art set and landmark pack.
  - New event interaction variant.
- Sprint B
  - New vehicle or handling profile.
  - New seasonal mission ladder.
- Sprint C
  - Performance pass and trailer capture.
  - Release, monitor, hotfix within 72 hours.

## KPI Gates Per Release

- 10-minute retention >= 70%
- Session length >= 25 minutes
- Share event rate >= 8%
- D7 retention >= 20%

## Content Backlog Structure

- `BiomePacks`: forest, coast, desert, snow.
- `EventPacks`: fishing, chopping, mining, camp variants.
- `CosmeticPacks`: body paint, wheel style, light trail.
