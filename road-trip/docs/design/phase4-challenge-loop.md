# Phase 4 Challenge Loop

## Weekly Challenge Spec

Panel toggle: `F3`

Default set:

- Distance challenge (`8 km`)
- Share challenge (`3 snapshot marks`)
- Mission challenge (`9 mission clears`)

## Reward Rules

- Challenge grants direct coins on completion.
- Completed flag prevents duplicate payout in same cycle.

## Trackable Share Challenges

- Share marker key: `P`
- Captures current journey and distance context.
- Feeds both KPI and challenge progress.

## Leaderboard Categories (next iteration)

- Longest no-collision distance
- Fastest benchmark test total score
- Most weekly challenges completed
