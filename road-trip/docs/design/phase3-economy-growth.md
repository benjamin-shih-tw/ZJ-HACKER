# Phase 3 Economy And Growth

## Resource Conversion Path

`drive and interact -> gather materials -> convert to coins at camp -> buy growth upgrades`

## Soft Economy

- Materials: `wood`, `stone`, `fish`, `food`
- Soft currency: `coins`
- Utility shard: `scrap` (future expansion slot)

## Growth Axes

- `handling` (steering-oriented)
- `efficiency` (reward and resource throughput)
- `style` (meta progression and cosmetic prestige)

## Upgrade Cost Rule

- Cost formula in prototype: `45 * currentLevel`
- Level cap per axis: `5`

## Pity System

- `fishFail >= 3` grants bonus fish on next success.
- `chopMiss` and `mineMiss` tracked for future adaptive assist.

## Balance Sheet v1

- Fishing success: +3 coins baseline.
- Chopping clear: +2 * axeLevel coins.
- Mining clear: +5 coins.
- Pulse reward: `5 + efficiencyLevel` coins.
