# Phase 0 Baseline

This folder defines the production baseline before feature-heavy implementation.

## Deliverables

- `version-scope.md`: MVP, Beta, and 1.0 scope boundaries.
- `project-structure.md`: canonical folder and document layout.
- `qa-acceptance-checklist.md`: release-gate checklist for feature, performance, UX, and KPI validation.

## Exit Criteria

Phase 0 is complete when:

1. Version scope is approved and frozen for current milestone.
2. Team uses the same folder and documentation structure.
3. QA checklist is used as the definition of done for all upcoming phases.
