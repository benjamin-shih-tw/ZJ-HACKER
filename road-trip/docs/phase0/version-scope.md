# Version Scope Definition

## MVP (Internal Playable)

Goal: verify the core loop works end-to-end.

Included:

- One playable vehicle with 3 drive presets.
- Procedural road + terrain streaming.
- Core POIs: fishing, chopping, mining, camping.
- Three-layer mission model (micro, mid, weekly).
- Basic growth/economy loop and local KPI tracking.

Excluded:

- Public leaderboard backend.
- Large content pack expansion.
- Full localization set.

## Beta (External Test Build)

Goal: validate retention and balancing with real players.

Included:

- Weekly challenge loop.
- Expanded mission pools and reward tuning.
- KPI panel + local snapshot export-ready format.
- Stability pass and onboarding improvements.

Success Targets:

- 10-minute retention >= 70%
- Average session length >= 25 minutes
- Mission completion rate >= 55%

## 1.0 (Ship Candidate)

Goal: release-ready game with repeatable content cadence.

Included:

- Stable core loop and economy balancing.
- Challenge and share loop polished.
- At least 4 biome-content packs (visual and event variations).
- Production-ready QA pass and release checklist.

Success Targets:

- 7-day retention >= 20%
- Share event rate >= 8%
- 30-day retention >= 8%
